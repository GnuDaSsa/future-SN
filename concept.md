# 성남 미래지도 기획 메모

## 핵심 구조

- 와이드 화면 중앙에 성남시를 상징하는 세로형 지도 실루엣을 배치한다.
- 성남시는 북부 원도심과 남부 분당/판교 축이 이어지는 긴 형태로 읽히도록 한다.
- 지도 내부는 기계적인 5등분보다 생활권처럼 자연스러운 유기적 경계로 5개 영역을 나눈다.
- 5개 분야는 재건축, 대기업/IT, 골목상권, 청년, 어르신이다.
- 사용자가 특정 영역을 클릭하면 해당 영역이 밝아진 뒤, 분야별 영상이 전체 화면으로 전환되어 재생된다.
- 영상 재생이 끝나면 팝업은 사라지지만 해당 영역은 계속 빛난다.
- 5개 영역이 모두 빛나면 내부 분할선과 라벨이 사라지고, 조각들이 하나의 외곽선과 하나의 빛으로 합쳐진다.
- 합체 이펙트 이후 "성남 미래지도가 완성된다."라는 최종 메시지가 나온다.

## 분야별 영상 방향

| 분야 | 영상 톤 | 핵심 장면 |
| --- | --- | --- |
| 재건축 | 상승감, 도시 재생 | 노후 주거지에서 새 건축 윤곽과 생활 인프라가 솟아오르는 장면 |
| 대기업/IT | 속도감, 데이터 흐름 | 판교 중심의 데이터 라인, 기업 네트워크, 연구 인력이 확장되는 장면 |
| 골목상권 | 온기, 생활 밀도 | 시장, 골목, 소상공인, 시민 동선이 다시 연결되는 장면 |
| 청년 | 실험, 기회 | 주거, 창업, 일자리, 문화 공간이 빠르게 전환되는 장면 |
| 어르신 | 안정감, 돌봄 | 건강, 복지, 이동 편의, 커뮤니티가 안전망처럼 이어지는 장면 |

## 현재 구현 상태 (2026-06-10 "Civic Mission Control" 재설계)

- 파일: `index.html` + `styles.css` + `js/` 모듈 8개 (구조와 디자인 규칙은 `DESIGN.md` 참조)
- 실제 영상 파일은 아직 연결하지 않고, 1초짜리 데모 슬롯으로 대체했다.
- 지도 실루엣은 `raqoon886/Local_HangJeongDong`의 경기도 행정동 GeoJSON에서 성남시 수정구, 중원구, 분당구 50개 행정동을 필터링한 뒤 외곽선만 추출해 반영했다 (지오메트리 잠금 유지).
- 인트로 발사 → 지도 → 클릭 시 전체화면 영상 → 영역 발광 → 5개 완료 후 합체 → 완성 메시지 → 60초 후 자동 리셋(전시 루프)을 상태 기계로 구현했다.
- 45초 무입력 시 어트랙트 모드("터치하여 시작" + 영역 쉬머)로 전환된다.
- 실제 영상이 완성되면 `js/config.js`의 `VIDEO_SOURCES`에 경로만 채우면 된다.

## 실제 영상 연결 시 교체 방향

```html
<video autoplay muted playsinline>
  <source src="videos/01-reconstruction.mp4" type="video/mp4">
</video>
```

권장 파일명:

- `videos/01-reconstruction.mp4`
- `videos/02-enterprise-it.mp4`
- `videos/03-local-market.mp4`
- `videos/04-youth.mp4`
